# CSV to Excel Converter (Python)
This project is a simple Python script that reads data from a CSV file, cleans the data, and converts it into an Excel-compatible file. It replaces missing values and prepares the data for easy viewing in spreadsheet applications like Microsoft Excel.

# Features
- Read data from CSV files
- Handle missing values automatically
- Clean and normalize data
- Convert CSV data into Excel-compatible format
- Simple and beginner-friendly Python code
- No external libraries required
